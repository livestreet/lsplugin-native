Плагин "Native" для LiveStreet 1.0
Текущая версия плагина: 1.2.2

Плагин делает сайт более "дружественным" к неавторизованным пользователям.
1. При попытке голосовать/добавить в избранное всплывает поп-ап с авторизацией
2. Форма комментариев в топике всегда раскрыта, можно написать коммент гостю и после отправки его перекинет на авторизацию с сохранением комментария
